# Non-stationary Combinatorial Bandits
Install the adaptiveRank package with its requirements. Run the commands:
```
python setup.py<br>
source adaptiveRank/bin/activate
```

Run a single experiment:
```
source adaptiveRank/bin/activate
bash script.sh
```

Run different experiments:
```
source adaptiveRank/bin/activate
python run.py
```
